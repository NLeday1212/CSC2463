Outline

          This project is the final project for the class CSC 2463, and was created by me, Nikolai Leday.
This project was created using javascript including the P5.js and Tone.js libraries. P5.js was used for
creating all of the graphical elements, while Tone.js was used for all of the sound elements. The final part
of the project was connecting the javascript to an Arduino controller. Connecting the Arduino was accomplished
by using a javascript library provided to the class from the professor which allowed the browser to read the
serial commands from the Arduino. The connection was able to work because of a program called the p5 serial
controller. The Javascript would send back commands to the arduino to change the number of lights that are on
which represents health.

Narrative Description

          There are three game states in Space Spelunker, the main menu screen, playing screen, and the game over
screen. The player is first introduced to the main menu screen, which shows a scrolling space background, the title
of the game, controls and how to play, and the points per entity destroyed. After starting the game, the player will
notice a time remaining and score text in the upper left hand corner. The player has three points of health, which is
represented with three heart images in the upper middle section of the scren. The player will be controlling a rocket ship
which can turn left or right, and fly foward. The player can also use the boost button to fly faster. Multiple asteroids
will spawn and begin flying around the screen at different angles and speeds. After a few seconds, asteroids of higher value
and aliens will spawn in regular intervals until the game ends. While the player is avoiding colliding with entities, they
will be firing lasers at aliens and asteroids to gain score.
          With ten, three, two, and one seconds remaining, a warning sound will play alerting the player to the
soon end of the game. Once the game has run out of time or the player has run out lives, the game will have ended and show
a game over screen. The game over screen shows the total score gained by the player, the number of asteroids destroyed, and aliens
killed. The player can then press the specific button to return to the main menu and start the game over again.
